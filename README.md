# FILORanked

FILORanked je aplikacija osmišljena za jednostavno praćenje, rangiranje i analizu vaših podataka. 
Projekt pruža intuitivno sučelje i napredne algoritme za sortiranje koji korisnicima omogućuju brz pregled i donošenje odluka na temelju preciznih ljestvica poretka.
